\# Reproduction Steps



This document describes the steps required to reproduce the experiments

presented in the GlobalAmBC-DRL paper.



1\. Install OMNeT++ (version 6.0 or later).

2\. Configure the INET framework.

3\. Compile the simulation project located in the `omnetpp/` directory.

4\. Run the baseline simulation scenario.

5\. Enable the GlobalAmBC-DRL control module and execute the DRL-enabled scenario.

6\. Use the scripts provided to collect and analyze performance metrics.
